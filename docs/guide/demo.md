::: tip 字体提示
在线演示运行于未安装公文字体的环境时，会用系统/开源字体兜底，观感与安装了仿宋_GB2312、方正小标宋等字体后略有差异。要精确还原，请在自己的环境安装公文字体或用 [`registerFont`](./fonts) 注入。
:::
