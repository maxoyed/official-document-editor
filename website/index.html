公文编辑器official-document-editor

纯前端、离线可用、headless 的中国党政机关公文编辑器。默认排版即符合 《党政机关公文格式》，支持 docx 无损往返、精确分页与打印，可在 Vue / React / 原生中开箱使用。

▶ 在线试用编辑器 Vue / React 示例 GitHub

核心能力

开箱即公文版式

A4 版心 156×225mm、每面 22 行；红头、标题、各级标题、正文、署名、版记、页码按 GB/T 9704 自动排布。

精确分页

headless 逐行分页引擎 + 编辑器内联实时分页（超长段落按行跨页断行）+ 白纸背景与页码，一键打印 / 导出 PDF。

docx 纯前端往返

命名样式无损还原；支持表格（含合并单元格）、图片、印章、版记分隔线。无需后端，离线可跑。

字体插槽

不分发商业字体；开源字体兜底 + registerFont 由使用方注入授权公文字体。

框架无关

headless 核心 + 薄封装：Vue v-model、React value/onChange，亦可原生使用。

TypeScript

全 TS 编写、类型完备，子路径按需引入：/spec、/pagination、/docx。

快速开始

pnpm add @maxoyed/ode-core
# Vue：  pnpm add @maxoyed/ode-vue vue
# React：pnpm add @maxoyed/ode-react react
@maxoyed/ode-core @maxoyed/ode-vue @maxoyed/ode-react
